📋 Lathund — Tidsrapport

Så här gör du

  1. Välj ett projekt och en kategori
  2. Ange ungefär hur stor andel av veckan det tog (i %)
  3. Klicka + Lägg till och upprepa tills du nått 100%
  4. Klicka ✓ Spara rapport

En grov uppskattning räcker — ingen exakthet krävs.

Kategorier

  • Nyutveckling — ny kod, nya features
  • Support / Maint. — buggar, driftsättning, teknisk skuld
  • SW / Backend för HW — SW-arbete kopplat till hårdvaruprodukter
  • Möten / Övrigt — möten, planering, dokumentation
  • Ledighet — väljs automatiskt vid Ledigt / Sjuk / VAB

Kombinera fritt

Lägg till en rad per projekt eller aktivitet. Exempel:

ProjektKategoriAndel
Thor ChargerSW / Backend för HW40%
FloorNyutveckling30%
MöteMöten / Övrigt20%
SjukLedighet10%

Tumregler

  • Fixat OTA för Thor? → Thor Charger + SW / Backend för HW
  • Byggt API för Floor? → Floor + SW / Backend för HW
  • Ledigt halva veckan? → t.ex. Sjuk 50% + Shield 50%
  • Mycket möten? → Möte + Möten / Övrigt för den andelen
